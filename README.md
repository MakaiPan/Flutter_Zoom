## Directory Structure

The project directory structure is as follows:
```
|--lib
    |--screens
    |--utils
    |--widgets
```


| **Directory** | **Explain**                   |
|---------------|-------------------------------|
| screens       | View layer directory          |
| utils         | Utilities                     |
| widgets       | Additional customized Widgets |
